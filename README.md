# SimpleGameforKidsTraining
Simple project to train kids with Claude Code.

## Game Examples

Each game is in its own directory with an `index_<DirName>.html` file:

- **CandyCloudRider/** - Goku's Candy Cloud Rider (side-scrolling shooter with candy theme)
- **SpaceFun/** - Nova Strike (space shooter)
- **Destroy&Discovery/** - (coming soon)
