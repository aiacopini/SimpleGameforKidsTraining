# Realm of Shadows (v1)

The original proof-of-concept for a Prince of Persia-inspired 2D action RPG with procedural graphics. This version served as the foundation for [RealmOfShadows2](../RealmOfShadows2/), the full production rewrite.

## How to Play

Open `index_RealmOfShadows.html` in any modern browser — no server required.

## Features

- 2D top-down grid-based gameplay
- Entity system with tile-based collision
- Game state machine
- Procedural Canvas graphics

## Tech

- Vanilla JavaScript + Canvas 2D
- 2 core files: `engine.js` (68KB) + `effects.js` (54KB)
