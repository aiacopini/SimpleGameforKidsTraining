# Dungeons of Drakenmoor

A D&D-inspired top-down dungeon crawler with 5 procedural levels, multiple character classes, turn-based combat, and grid-based exploration.

## How to Play

Open `index.html` in any modern browser â€” no server required.

## Features

- Character creation with multiple classes
- 5 dungeon levels with varied enemy types
- Turn-based combat with dice rolls
- Inventory system and boss encounters
- Particle effects and medieval audio (Web Audio API)
- All character sprites drawn procedurally on Canvas

## Tech

- Vanilla JavaScript + Canvas 2D
- Modular architecture (11 JS files)
- Key modules: `game.js` (loop), `sprites.js` (procedural art), `combat.js` (turn-based), `levels.js` (generation)
