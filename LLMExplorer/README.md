# LLM Explorer

An interactive educational tool that visualizes how Large Language Models work. Walks through 7 stages of LLM inference with explanations, animations, and quizzes — aimed at ages 14–16.

## Features

- 7-stage inference pipeline: Tokenization, Embedding, Self-Attention, Feed-Forward, Output Distribution, Tool Use, Token Generation
- Multi-language support (English, Italian, French, Spanish)
- Interactive quizzes and knowledge checks
- Side-by-side prompt comparison mode
- Probability distribution charts (Recharts)

## Tech

- React 18 (single functional component)
- Recharts for bar chart visualizations
- Built-in i18n (no external library)
- Requires React tooling (Vite or CRA) to run — not a standalone HTML file
