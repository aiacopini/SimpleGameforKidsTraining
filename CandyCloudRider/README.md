# Candy Cloud Rider

A Dragon Ball Z-inspired kid-friendly platformer where Goku rides the magic Nimbus cloud through candy-themed worlds. Instead of combat attacks, Goku throws cookies and faces cupcake bosses.

## How to Play

Open `index_CandyCloudRider.html` in any modern browser — no server required.

## Features

- 3 levels with increasing difficulty
- Difficulty settings (Easy / Normal / Hard)
- Boss fights with candy-themed enemies
- Built-in tutorial system
- Progress saved to localStorage

## Tech

- Vanilla JavaScript + Canvas 2D
- Single HTML file (~213KB)
- All graphics drawn procedurally — no image assets
